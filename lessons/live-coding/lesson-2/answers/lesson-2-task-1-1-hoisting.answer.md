# Ответ к задаче 1.1: Hoisting

**Ожидаемый вывод:**

```
undefined
```

**Объяснение:**
Переменная `a` объявлена с помощью `var`, поэтому её объявление поднимается вверх функции (hoisting), но инициализация происходит на месте. До строки `var a = 10;` значение `a` — `undefined`.
