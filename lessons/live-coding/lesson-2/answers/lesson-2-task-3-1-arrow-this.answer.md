# Ответ к задаче 3.1: this в стрелочной функции

**Ожидаемый вывод:**

```
Hi, I'm undefined
```

**Объяснение:**
Стрелочные функции не имеют собственного `this` и берут его из внешнего контекста. В данном случае это глобальный объект, где `name` не определено.
